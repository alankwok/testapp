【手機】Motorola Milestone 3 中庸中價卻有賣點，商務合用

其實不是什麼搶先試玩， 在國外Motoroal milestone 3 老早已經發售， 上兩星期才到手是因為mo記商業策略的原故， 香港到10月21日才正式發售， 當中有幾項重點值得評價一下， 而其中(前文)Motorola 充滿誠意的新介面 Moto BLUR， 於上星期已組寫過了， 今天再談談其他。

看上Milestone 3 的用家， 我大膽地說這必然是因為那QWERTY鍵盤， 於 iPhone 大熱的今天， 有鍵盤的智能手機真是不多選擇， 如果要有一個好的系統配合， 在今天來說可能只有Android 的份兒。

::



玩家對 QWERTY鍵盤的需求， 就必然是
建基於用全鍵盤確實比模擬鍵盤打字快。 十年八年前我十分鍾情於 Palm 的鍵盤， 因為打字實在超爽， 後來時代轉成流行milestone 3 這類橫向滑蓋式 QWERTY鍵盤， 就令一眾廠商都一同朝這方向發展。


鍵盤超讚!!!
Milestone 這橫向滑蓋式鍵盤的按鍵算是很大尺寸， 用我肥大的指頭都沒有因而錯按旁邊的鍵， 按下的時候按鍵的反彈力度亦很不錯， 更有一種實在的感覺， 雖不是最好的那種， 但亦是中上水平的設計。再說， 2代的鍵盤是一整塊膠的設計， 今次的3代剛改成每顆獨立按鍵，  按鍵的時候感覺更爽!


專屬的數字鍵提高方便程度
Milestone 3 此次設計的 QWERTY鍵盤有一個超級讚的點子， 就是它們將舊有設計的數字鍵和字母鍵分開了， 然後於上方加多了一行專屬的數字鍵， 整個鍵盤現在共有五行的按鍵， 玩家輸入數字的時候不需要再切換不同的模式， 大大節省了時間並提高了方便程度，  鍵盤的面積亦比2代的時候因而增大了。而在空格鍵的左邊保留了Milestone 2 增加的搜索鍵， 這樣在使用鍵盤時就方便多。鍵盤更加印了倉頡碼， 如我們香港人或是台灣的朋友打中文的時間就更見方便直接。


螢幕尺寸足夠平常使用
↓ 螢幕方面， 4 吋 及540x960 qHD 解析度， 大小及解析算是適中， 因為最近 htc 都已經推出螢幕足足有 4.7吋的 sensation XL， 如果要數算大螢幕的Android就一定不是 Milestone 3， 但 M3 的螢幕尺寸實在絕對足夠平常使用， 當然地再大一點就更好啦。


整體重量要減磅
但整體重量方面我就希望Motorola 多加留意， 現時的 M3 實在重了一點， 男玩家使用都尚算可以接受， 但如果可以減掉15%左右的重量就更會吸引女玩家的留意。

1080p 高清拍攝

電磁容量一般
↓  打開後蓋看見內裡的電磁 ， 因為我早早拿了工程測試機來玩， 所以電磁都沒有印上容量， 當大家現在買的時候， 則應該是 1500mHz， 使用情況與市面上一般的大至相同， 若一整天不停上網的話， 就應該不足夠， 早上上班， 可能六時放工前就要充一下電， 預備晚上出外吃飯時可以有電把玩。



::
數據上分析不太輸蝕
如果有聽我的聲音博客節目 玩樂手記ITM， 都會知道我平常不太著重數據上的分析， 只會在意於實製使用上的便利性， 但今次有點例外， 因為於其他網上討論區對milestone 3 有很多差勁評論， 所以我都安裝了一兩個brenchmark 的軟件抓一點數據給大家看。

就先看 AnTuTu 出來的， MileStone 3 得分4942， 次於 Samsung Galaxy S II 及LG Optimus 2X 之後， 總評分看比例上 Milestone 3 比 Galaxy S II 整體慢了 10% 左右，  主要是 Ram 與及CPU Integer 方面多一點差距， 但 2D 及3D 圖像處理其實不太輸蝕。

Milestone 3 選用的是 TI OMAP 4430 1GHz 雙核心處理器，多媒體支援性最見突出，能播放 1080p 全高清影片及直播 RMVB 及 H264 mp4 檔案，  "直播 RMVB"這算是 oMAP 4430 的獨有秘技。相對地milestone 3 處理 3D 圖像方面就有一點缺失， 但這本來就並不是 milestone 的賣點， 從第1代開始就已經不是。


整體速度尚算合理夠用
數字不再提， 實製使用時其實尚算流暢， 特別於 Home Screen 介面之間遊走的時候， 基本上的是順暢非常， 沒有不順的情況。但當要開啟一些軟件時， 則間歇性地有一次出現半點遲緩， 可能始終Milestone 3 的雙核心都只是1Ghz 的設定， 所以開啟軟件就出現了這種情況， 但整體速度上尚算合理夠用。

軟件合商務人仕使用
軟件方面有一些軟賣點...... Milestone 3 內置可編輯的 Quick Office 文書處理軟件， 相信這是鍵盤機的必用軟件， 零售價都要數百元港幣， 現在買機就已經送了， 算是幫忙省下小小金錢。另外亦有一項很多人希望使用， 但又是很少廠商會提供的通話錄音功能， 可以在通話當中隨時秘密錄取通話雙方的內容， 一按即錄。

Quick Office與通話錄音功能兩個都是商務人仕合用又實用的軟件。

::
總結：
中庸中價，卻有鍵盤作獨特賣點， 商務人仕合用， motorola 誠意實用產品。
