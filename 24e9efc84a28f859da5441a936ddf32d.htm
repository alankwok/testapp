【教學】最便宜HK$9 購買自己的網址 Domain Name，建立個人品牌，避免YahooBlog 搬家麻煩，不再寄人籬下


Yahoo Blog 搬家的確麻煩，我亦不明白為何仍然有Blogger 貪一時方，將自己的心血搬去另一平台，由一隻沉船搬去另一隻小船，而這篇教學會教大家一勞永逸的解決方案， 就是擁有自己的Blog 網址，用最便宜的方法 HK$9 購買自己的Domain Name 自設網站名稱，建立個人品牌，不再寄人籬下。日後再遇沉船事件都不再流失讀者。
這篇教學會教大家下列的事情:
- 開立全球最大網址銷售商Godaddy  戶口
- 找尋自己喜歡的網址
- 找Godaddy 優惠碼節省金錢
- 付款時留意事項



注意! 首先說好， 請跟足下列的次序先開立戶口才能找到輸入優惠碼的位置。

(1) 去godaddy.com ， 右上角有一個 "Create an account" 鍵， 在這先要登記一個新戶口，容後用來付費買網址。

詳細步驟不寫了，重點留意事項如下: 登記時，Bill info 及有關信用咭的資料要正確、真實，因為付款時要跟銀行核對用。

聯絡資料及技術員（Contact Info & Technical support) 的資料可以隨意填寫，因為他們都查不到，要留意這兩類資料跟會根據美國法例原故於互聯網上公開，所以自行考慮怎填吧，但重點是為了日後真的能收到各樣通知，所以電郵地址必要真實能用。(到內裡時要求你填Zip Code， 但香港是沒有的， 填0000 騙過系統便可以了。) ▼





(2) 開設了戶口， 可以直接按Logo 回到godaddy.com 的主頁， 你會見到一個很大的空格給你搜尋網址用， 於當中填入你想買的名字便可以， 如果已經沒有了，請多試幾個別的。.com 人們最易記得， .org / .net 那些其他的， 大家唔有那麼直接記住，所以都是用 .com 的網址比較好。▼





(3) 如果你填的網址仍可選購， 這裡智出現 "xxxxx is available"字句， 這時按右面的 "Add" 鍵加入購物車便可，同時 Add 鍵會轉成 Added。▼





(4) Added 鍵上方有另一個綠色鍵寫著 Continue to Checkout > ，按它一下去另一付款結賬的步驟吧。▼





(5) 跳到這裡Godaddy 會問你要不要 Hosting 網頁儲存、電郵地址之類的服務， 我們會將Blog 免費存於Blogspot (Blogger.com) 當中， 所以全部都不要了，直接按下方的灰色 No， thanks 鍵又再跳到另一付款結賬的步驟便可以。▼





(6) 如果你是舊的Godaddy客戶便會有可能出現這個提示你其他舊有網址要到期付年費的畫面， 如果沒有出現便跳轉下一步驟吧。▼





(7) 到這裡都最重要的部份：▼

(i) 首先， 我們要用最優惠的價錢去買這個網址，便需要找到一個優惠碼 (promo source code)，這個優惠碼於網上不難找到，你可以到facebook 公開找我問問最新的優惠碼，又或是Google "godaddy promo code" 都可以。

(ii) Godaddy 優惠碼都是一年一年地給節扣的，為了用盡優惠，所以我們於 godaddy 剛才的付款頁面中，於網址右面的購買年份由預設的兩年改為"一年"，那麼下年我們再找另一個優惠碼便可節省更多了。

(iii) 找到優惠碼之後，於godaddy 付款頁面中下方的位置找找 "HAVE A PROMO CODE?" 字句，按一下變成方格，再填入剛找到的優惠碼，例如今日找到的 "99COMS" (8 Oct 2013) ，就可以特價第一年 US$0.99 購買網址一個。加上ICANN 的固定登記費用US$0.18之後，大約就是HK$9.12 一年。看清楚填入優惠碼後對價錢的影響及變化，Godaddy 不時會限制優惠購買的條件，你找的時候網站寫著US$0.99一個，到幾天後才用可能已經變成US$5.99 或完全用不到都不足為奇。如果優惠碼不對勁便再多找幾次吧，Godaddy 的優惠多的是。

(iv) 再按右面橙色的 Proceed to checkout 鍵跳去作最後確認完成付款。





(8)  到真正的最後步驟，於頁面中間選擇信用咭付款，早前已經填好了資料讓銀行批核，不用再填，未填的，現在墳寫也沒有問題，成功付款後網址便即時使用。▼

但這裡有一個常見的問題， Godaddy 系統會不接受你的付款， 並彈出一句 "There was a problem processing your transaction. Please verify your payment information or use an alternate form of payment.“ ，但又沒有提到是什麼問題，這情況我早前都預過，當時的原因是 Godaddy 將 Promo Code 的條件改了，由原本國際地區均可使用，改為只有美國區可用。所以導致付款系統沒有反對Promo Code 的節扣優惠，但核對買家地址時及IP 位置時最終不獲系統接納，不接受購買。

當時的解決方法就只有一個，就是再找另一個 Godaddy Promo Code 是供國際地區使用的， （如前段所說，Google 一下就有了）， 如果沒有US$0.99的，便要退而求其次用 US$1.99 / 3.99.....等。





(9) 另外亦會收到電郵再次確認。▼



(10) 購買網址的步驟全部完成了。

P.S.
(a) 將Godaddy 買的網址停泊到 Blogspot 的設定技巧會另文介紹。

(b) 將Yahoo Blog 的備份檔無痛輸入到blogspot 的技巧我會另文介紹，我跟程式員於設計了一個小工具放 Facebook 幫忙轉檔，這個跟大家於台灣見到的工具有很大的分別。我們的解決了 Yahoo Blog 分類不能轉移的問題，亦解決了太多文章及相片便入不了BlogSpot 的問題。
