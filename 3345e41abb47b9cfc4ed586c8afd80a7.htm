【美。男士美容】Braun x HUGO BOSS 限量版電鬚刨試用報





上兩星期收到品牌公司送贈的禮物 Braun x HUGO BOSS 限量版電鬚刨，第二天的早上其實已經開始試用， 到了寫博文的今天都用了差多兩個星期， 應該有足夠的體驗可以介紹一下 Braun 這個電鬚刨吧。

先說明一下， 我自己於三數年前開始就已經用 Braun 的電鬚刨， 第一個是朋友送的， 都是中階的型號，有自動清洗器的那種，用了三兩年時間， 正好於個多月前到期收工了， 我亦沒有專誠跟廠商查問保養的事情，我想...反正用了三年時間，到期收工是正常吧， 所以我亦馬上到了家樓下的電器店自費買一個新的用。而更幸運的是上星期收到公關朋友代品牌送贈的這個於父親節前推出的 Braun x HUGO BOSS 限量版電鬚刨， 可就是給我先測試及考慮用它作為禮物。

Braun 其實即是「百靈牌」， 如果是十年八年前，大家都會常常聽到「百靈牌」贊助一些電視的遊戲節目等，所以品牌的名字對我這上70年來說是已經「入晒腦」。但今次「百靈牌」跟 Hugo Boss 的合作， 如果源用中文的名字就好像令整件合作老土了。所以...都是乖乖的說成《 Braun x HUGO BOSS》。


《功能特點》
Braun 今次跟HUGO BOSS 合作的電鬚刨， 其實就是用上了Braun 他們頂級的 7系型號790cc 電鬚刨作藍本， 擁有高科技功能， 每次觸動能產生10，000次微細震動的聲波剃刀震動技術， 與及感應貼面彈性刀頭上的三級長短鬚修剪刀，快速、徹底、舒適的完美剃鬚體驗貼面刀頭設計。

功能上其實跟同品牌的3系5系相比，舒適程度反而很接近，鋼網刀頭貼著面的順滑感覺相同。而7系最大分別在於剃鬚所需時間大大減少，面上的鬚根亦剃得更為乾淨，最明顯的是脛上逆向生長的鬚子，用這Braun x Hugo Boss 電鬚刨處理會最容易感受到它的厲害，我認為這更是取決於實用性上的最大賣點。


《HUGO BOSS設計合作》
再由HUGO BOSS方面負責將鬚刨設計成高雅的藝術品般，基本上外觀全黑色鋼琴面設計當然地吸引，每朝早拿著剃鬚，心理上都已經享受著。還有那由Hugo Boss 處理， 獨一無二人手縫製的旅行用皮套，絕對地型格。


《配套齊全》
至於那個充電及清潔專用插座， 其實就是一個完善的自動清潔系統， 用家只要將電鬚刨插進去，插座便會自己進行清潔、殺菌、潤滑和烘乾等程序，保持刀頭衞生。這個插座相關的真的沒有什麼可以多提，就是一個「方便」。


《價錢尊貴》
而Braun x HUGO BOSS 電鬚刨唯一的問題是， 此電鬚刨定位於高階功能及奢華級別，所以價錢定於 HK$3998， 絕對不是一般人能輕鬆接受，但這亦令這Braun x Hugo Boss 電鬚刨更顯尊貴。你的父親節禮物想好了沒有?? 不妨留意這個Braun電鬚刨。都是那一句，選購與否，自行決定。
