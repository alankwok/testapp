【音響】Ultimate Ears 600vi 隔音耳機簡評

從iPhone 3Gs 轉到  iPhone ４的時候，我便將我的音樂庫獨立了出來，不再將家中CD轉錄出來的mp3檔抄到 iPhone 4 當中，始終我對於數碼玩具都是一個朝三暮四的人，那怕手中有了 iPhone 4 ， 只是聽到傳訊iPhone 5 的拍攝功能可能達800 萬像素，就已經決定要買了。而且　iTunes 抄錄歌曲真的超級的煩人，所以只好多買一部 iPod儲存心愛音樂，iPhone 嘛就獨立運作好了，不用每次傳輸一下備份一下數據的時候都帶一點煩氣，因為次次都好像沒太大信心似的，過程總是覺得有點問題一樣，費時失事。對於音樂我多 喜愛 jazz、國語類及 hiphop， 當中以 jazz 作首選，由其喜愛比較早期的舊曲目，頗有味道的，hiphop 嘛，當中的文化更是值得細看。差不多一個月前公關朋友送來了 Ultimate Ears 600vi 隔音耳機給我一試。是的，快一個月了，文章拖了很久都未有寫得出（美女公關麻煩妳了～），可能是太享受音樂的源故吧! hahaha今天就談一談此Ultimate Ears 600vi 隔音耳機。

::




::

Ultimate Ears 出名是專為客人訂造入耳式耳機，而耳機的品質也帶給用家錄音室級的鑑聽表現及隔音功能。以往售價更是動輒過千港元，大眾亦視UE為較為高階的一個品牌，而 Logitech則於2008年將 Ultimate Ears正式收購， 並提供了專業的工程技術協助，大大提高了技術上的支援。因此今天 UE 就帶來了我一套600 vi 作試用，我就當然馬上答應喇，誰不想有得玩有得享受呢！？　haha

::



::

基本標準的 3.5 mm DC jack 鍍金接頭 ↓



::

包括了耳機及mic 的線控裝， iPhone 、iPod 及 iPad 都合用， 其實大部份的手機都看似可以， 唯獨是網上有其他用家提到 SE 的手機就未能配合， 但有代查證， 知道的亦可分享一下  ↓



::

600 vi 的「vi」就是代表支援全數擁有3.5mm介面的手機， 那當然地兼容當時得令的 iPhone，而且具備話音功能的，即是大家可以用 600 vi來接駁來電，不用又再拿掉耳機再接電話。方便程度相信不用多談了。

另一小弟喜歡的特點，就是Ultimate Ears 原裝就來了5+2 共７對的耳塞，有XXS、XS、S、M、L 各尺寸，而當中有兩對是 Comply 棉製造，就是大家用的時候要先擠壓一下，令它變少，再放進耳朵中，待Comply 棉回復了原來尺寸的時候就正好貼實耳內，　此有趣的特性就形成了600vi 有著絕好的隔音效果。就是跟 UE 一向的高水準製作一樣。

::

600 vi 原廠跟來的Comply 棉， 左面的擠了數下， 右面的就沒有了 ↓



::

沒有安裝好 comply棉的 UE 600 vi， 左右兩邊的外殼顏色不同， 一紅一黑， 方便分辨左右耳使用 ↓



::

原廠一套共七對耳棉， 其實5對是入耳式軟膠耳塞， 尺寸分別是 XXS、XS、S、M、L ↓



::

另外兩對就是我個人比較喜用的comply棉  ↓



::

附帶的保護盒， 方便將耳機及附件都一拼收拾好， 很貼心的! 亦令市面上很多品牌都一一學習了此安排。



::

音質方面，我用了 comply 棉來配合測試了 Ultimate Ears 600vi 使用的單一單元，原以為音質方面相比市面上的其他品牌會差了一段，但原來經UE自家調校過高中頻音色後，600 vi 卻有著廣闊的範圍，令整體風格表現不俗。由其是試聽女聲的時候，音質更是清脆細緻，例如謝安琪， 王宛之偏高中音聲底之歌手，所以若收聽我個人喜好的jazz 爵士音樂都會表現不俗， 此乃600 vi 帶給了我一點點驚喜。沒想過單一單元亦能有此般的音樂層次感，很舒服， 很自然。

但對於我個人的另外要求 hip hop 嘛，先天性就是另類一點的要求，在此不免強評論。

 

總結:

$8xx 的價錢，算是 UE 耳機的入門級別，而作為入門級別，此 600vi 則是優先選擇的一對，Comply 棉隔音再加上支援手機來電，方便程度頗高的、頗實用的～　整體性價比不錯的一對耳機
