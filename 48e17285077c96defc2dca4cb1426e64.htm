Nokia Lumia 800 博客收Phone聚會 & 發佈派對 x 鄭秀文 x 陳喚仁

早前獲Nokia 邀請出席新品Lumia 800 的特別聚會，有什麼特別呢？ 就是因為他們這次的出席名單不單包括了Blogger 們參與，而且邀請了一些科技網站的記者們一同出席，雖說是兩組不同的人，但其實大致上都是相熟，始終自己是主力於科技產品的文章。

聚會當晚我們一共十數人於銅鑼灣Nokia旗艦店的閣樓相會，約八時許就先開始了小吃及自由活動的時段，我因為有另一博客活動要參與，所以只好八時多才能到達，感謝大家的體諒，希望沒有打擾了大家。





::

我一來到公關朋友不用多說就已經將預備給我的Lumia 800 放於枱子上給， 精美的包裝一大盒的，裡頭更專誠印著字句說明這次是全球中預先成功收到 Nokia Lumia 800 的一群，頗有優越感。



::

同一時間 Nokia 的同事就已經招集大家坐定定， 好方便他們向大家講解Nokia Lumia 800 的各項特點。讓我粗略的介紹一下當晚的情況。



::



::

首先在於介面及系統上， Nokia Lumia 800 用上了 microsoft 的 windows Phone 7.5 系統。 7.5 版本除了有功能上的改善之外， 對我們中文用家來說就是主要加入繁簡體的中文介面， 這個在中港台三地是必需的， 如今7.5 版本可以滿足大家的基本需要。



::

再看 Nokia Lumia 800 的主頁畫面就已經發現它有著 Nokia 的特色， 一打開已經見到預裝了好幾個本地化的軟件， 對於港人的需要， Nokia 一向都是頗細心的，這更是 Nokia 的強項。



::

再來就介紹了 Window Phone 系統本身附有的文書處理軟件， Microsoft 的主打功能 - Office Mobile 。



::

Windows Phone 裡面的市集主要分為兩部份， 一部是應用軟件， 另一部份就是 Xbox 的遊戲。其實於日後出貨的時候， market Place 當中會多了一項 Nokia Collection， 內裡收集了Nokia 專有的軟件， 有一些合作可能其他品牌的 WP7 機種就不會有附帶了， 這種製造專屬優惠的推銷方式其實都是很平常。 而上述的功能相關的評測我會另開文章評評， 大家留意吧。太忙了沒有辦法即時寫好。



::


正當Nokia 同事初步介紹了Window Phone 7.5 的時候， 是晚活動亦到了尾聲， 大家已經將手上的 Lumia 800 打包中， 因為是次Lumia 800 就是給我們帶回家中玩過夠， 整晚的時間其實就在聽聽玩玩食食。我於早前其實都已經將 Nokia Lumia 800 手機規格、價錢 有關的資料於本blog 的新聞欄刊出， 大家可以先看。

::
收Phone會完成了的第二天，  Nokia 其實有一個給傳統傳好的發佈會於the Grand Stage 即是上環西港城大舞臺飯店舉行 ， 我與一眾友好都有出席這個活動， 雖然地方有點不方便， 但活動當中有兩個表演嘉賓極之吸引， 其中一個就是Sammi 鄭秀文。



::



::

而另一個就是2011 年於香港紅到發紫的 老師 Hanji 陳喚仁， 他們兩人都分別有人作出音樂表演。



::

Hanji 老師 表演拿手的新派輕 Jazz 樂



::

會場中有一些小位置展出主角 Nokia Lumia 800 與及首批推廣送贈Monster beats 耳機



::

另一個位置預備了讓來賓試玩 Xbox Live 遊戲，  可能是我來晚了， 人群早就已經試玩完畢， 拍照時沒有人在玩這個項目。



::

但試monster beats 耳機的位置卻人頭湧湧， 這倒是奇怪的， 可能到場的朋友比較成熟吧!? 沒有太多人試遊戲部份。



::

現場我見到的 monster beats 耳機有白色與及藍色可供選擇， 但服務員提到發售時備有黑、藍及粉紅三色選擇， 售價與推出日期仍然未定。
