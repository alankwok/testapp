清涼薄荷味 Yahoo mail beta – minty

早前出席了 Yahoo mail beta 的博客聚會，  yahoo mail beta 有一個新名字叫 "minty"， 雖然就是 beta 的 mail 功能，實際上來說是並與時並進地revamp 一次。小弟獨立地看 yahoo 的mail 功能， 其實一向有好印象，唯記得好幾年前將 web mail 的介面作了一個大改變，直接對當時估了市場大份的email 軟件學習及參考，令 yahoo web mail 看上來與那軟件有九成像樣，因而大大方便了一批慣了PC 軟件的用家，亦因而賺回了不少口碑及用家數。

↓ Yahoo 新revamp 的電郵服務跑回轉頭自稱為 mail beta， 小弟倒覺得沒有所謂， 表達了是一個重新設計的版本就是了。



加了很多新功能，而且更加了一個口號  "faster.safer.easier"  更高速。更安全。更易用


今次 minty 的介面更貼近一個 web app 的模式， 算是整體上回到internet 的世界， 因為上次的確實太像一個軟件， 但可惜那軟件是不喜歡用的， 但到了今天大家對internet上的服務用多了，  Minty 轉回一般網民都慣了使用的 web app 介面倒是更易習慣。



說Minty 的新功能增多了，  這就是其中一項， 用戶可以方便地放大查看電郵內的相片， 更可以快捷地轉到另一張去。


↓ yahoo minty mail beta 有另一個新功能， 就是用戶可以免費傳國際短訊 SMS 給其他手機號碼的朋友， 而朋友回覆的時候就集中到 yahoo minty 當中， 亦算幫助中央管理了另一項大家常用的信息。


根據yahoo 方面的朋友回覆， 這服務是免費， 於現在這階段更沒有限制免費短信的正常使用。但是管理員一旦發現費短信包含垃圾郵件內容，他們將控制數量的使用。而這是指廣泛性的控制還是對個別戶口的控制， 這就是仍未知道。



Yahoo minty 的另一功能就是能夠與其他手機平台直接進行視像通話， 當晚小弟看著如下圖的通話情況， 看著它整體上都是能流暢地使用，  效果實是不錯， 日後用家就可以隨時隨地用任何手機 (如iPhone、iPad、Android、Nokia、Samsung )， 又或是電腦上網收發yahoo minty 電郵和視像通話及線上聊天。



↓ 而今次yahoo mail beta 這版本更加入了 social media 的整合， 大家可以加入 Twitter 及 facebook 的 states 更新到 yahoo mail 的版面， 一次性地看到身邊朋友的最新分享。但可惜的是未有新浪微博的整合， 相信這是個非技術性的難題呢。





最後一提， Yahoo Minty 是現在市場上唯一提供無限儲存的電郵信箱，可以將你電郵信箱變身網路硬碟，無限儲存你的相片、影片、文件和檔案，方便隨時存取。希望試用yahoo mail 新beta (revamp) 的網友可到下列網址 http://hk.features.mail.yahoo.com/
