【手機】Acer neoTouch 簡單測試


首先感謝公關公司借了一部印著 1010 字樣的 Acer neoTouch 我玩了差不多一個月時間，  但筆者最近事忙， 只能作一點基本的簡單評論， 但它仍能令我對 Acer 這品牌多了正面的觀感。




Acer neoTouch的尺寸基本上跟iPhone 一樣大小， 用慣iPhone 的都不會覺得太大， 這是我拿在手中的第一個感覺。

-



量度一下螢幕對角的尺寸足足有9.8cm / 3.8吋闊， 實在巨大! 因為iPhone 都只是有 9cm / 約3.65吋左右， 這尺寸十分能夠滿足一眾喜歡用手機看電影或上網的用家。

-



而且Acer neoTouch 使用了Qualcomm Snapdragon QSD8250 1GHz，  因此運行軟件時極為順暢!!

但有一個不知歸咎於Acer / microsoft， 又或是我手指頭的問題， 因為於我基本地使用螢幕觸控時， 覺得實在不易習慣， 有著螢幕好像不太靈敏的感覺， 更不時有拉不動的情況出現。

-



但另一方面 neoTouch 有一個優點是它支援 MicroSD 記憶咭， 更內置有32GB的容量。而相片中後方的就是SIM 咭的插槽， 但可惜拆掉SIM 咭時會有困難， 要用膠紙貼著才能拿出來， 對一些經常換咭的朋友會不方便。

-





我再試用自動模式在正常室光下拍攝一張彩圖， 左面有日期的照片是acer neoTouch 所拍， 拍照質素不錯， 當然地比右面那張用 iPhone 所拍的更為清晰， 因為neotouch 是內置 500萬像素鏡頭! 成色當然比iPhone 的好。

-



有一個值得嘉許的就是 Acer / Microsoft 將一些如Blogger， Flickr， youtube， Facebook 等等social media 社交媒體的網站連結加進了neoTouch 成為按鍵方便大家使用， 但可惜沒有提供 Twitter 呢! 所以朋友們要自行安裝了。

-

但有一樣怪怪的事， 於新的 Windows mobile 6.5 版本中Microsoft加入了 Windows Marketplace for Mobile，  但當我一按下去的時侯， 系統會自己查看marketplace 是否最新版本， 而且要花幾分鐘的時間去慢慢查， 若按下"結束"鍵又會跳回主畫面， 所以我不知為何microsoft 有這樣設計， marketplace 不就是賣軟件的地方嗎? 用家就是想買什麼新的才會有意慾按進去看， 但又怎會花幾分鐘的時間等你的更新? 什麼意慾都會冷卻吧？



-

總結:

neoTouch 只需要HK$3680 是Acer 有誠意的作品， 一部這樣性價比的windows phone 真的十分吸引， 如讀者習慣使用WM ， 那可以聖誕節送給自己作禮物。但 Windows Phone  (舊名 Windows Mobile) 這系統不是我杯茶。
