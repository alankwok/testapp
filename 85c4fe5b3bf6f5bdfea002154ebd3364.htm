【配件】SHOT NOTE - 手寫藝術數碼化

聖誕節前與友人飲酒暢談，友人帶來了小禮物，說是受了另一公關朋友所托要送給我試用。小禮物就是於日本十分有名又暢銷的不平凡記事簿 Shot Note。 Shot Note 的不平凡是因為它不單是一本記事簿，而且更會配合自家的手機軟件同時使用。


[手寫記事、數碼記事之優點與缺點]
從前， 日常生活中我們會把鎖碎事記在記事簿上，優點就是可以隨心寫下文字，或多畫個公仔，表達到重點，好讓日後番查待辦的事項， 而且整個記錄流程能快捷完成， 因為不用慢慢輸入資料嘛。但手寫亦有缺點， 就是當遇到一些趕緊的時候， 我們亦只能選擇隨手拿起紙筆記事， 用了的記事貼更會隨處亂放， 好容易便會不翼而飛， 日後要找回資料或將之分類亦都很費力氣。

幸好現在人們大多數
都有一部智能電話在手， 更習慣了將資料集中輸入到手機當中， 好方便安排日常的生活。但可惜輸入資料並不是一件省時的工作， 往往用回手寫的方式會更為實際快捷及靈活，

Shot Note 的定位就是要希望補足及令到兩個不同的記事方式能互相配合起來， Shot Note 的主要功能就是協助我們將手寫出來的資料， 簡便地記錄一個影像到手機及雲端應用當中， 把兩個記事方式的優點共融使用， 令用家得到最大的好處。


[Shot Note 硬件設定]
↓ 先看看Shot note 究竟包括了什麼東西在內? 我收到的是紅色高級尼龍質料再用黃線縫上， 外表平實，  book style左右開的方式， 雖然質料不錯， 但紅色實在不太合我這種大男人使用， 如果是黑色的會好一點， 若然是真皮的就更為高貴。





↓ 這紅色Shot note 套的內格則左右各有一個膠套， 方便用家插入一些紙張等東西， 中間則掛著兩條作書籤功能的繩子， 而且分別是黃色及金色， 令使用時可辨明兩者的分別， 細心的設計!





↓ 另有一本只有二十多頁， 薄薄的、左右翻的單行簿， 是普通的， 沒有什麼特別。





↓ 單行簿尺吋剛好可以插到膠套中， 用完了就換一本新。





↓ 另外Shot note 有一本記事簿及一枝筆在內。這本記事簿就不同了， 是Shot note 的半個靈魂， 編號 No. 9101 應該是代表這本簿的型號及尺寸， 而簿面上的圖示就說明了Shot Note 整個方案的使用方式 -  (1) 用iPhone 拍下手繪圖像  >  (2) 用iPhone Shot Note 軟件的OCR 功能記錄日期 > (3) 再選擇上傳到Evernote 或用電郵分享





↓ 打開記事簿看看內裡的情況， 發現是一本格仔簿， 4個角位分別有一個標記， 就是給iPhone /iPad 上的Shot Note 軟件作定位之用。有必要時可撕下一兩張空白的放到另附的膠套中獨立安放。





↓ 再看看附上的筆， 其實只是普通的可換芯原子筆。



↓ 但筆桿頂部有一軟膠， 用家可以用來使用於iPad 及iPhone 等觸控螢幕， 靈敏度是不錯的， 亦沒有接觸不良的情況。




[Shot Note 軟件功能]
而Shot Note 的另一半的靈魂就是iPhone / iPad 上的軟件， 用家使用了上述的特製記事簿畫了圖， 再用iPhone App 拍一下， 影像就會被存起來作記錄。早前我打開雜誌抄了幾幅漫畫， 再對一對4個角定位， 就拍了進Shot Note App 當中， Shot Note 更會自動作梯型校正。













圖像拍好了， Shot Note App 就會自動到了設定畫面， 用家改改Tiltle 便可以。而Tag 部份就已經預設成"Shot note"， 自己新增多幾個亦可， Tag 分類主要是方便於容後的時間再翻查記錄， 其實即是自設一些關鍵字囉。





再跳轉看Shot Note App 的主畫面， 圖像都因為OCR 對記事簿上設定的日期作了排列及設定， 資料清清楚楚一目了言。













我覺得圖像到了Shot Note App 後的對比度、色彩度看起來有點怪怪的， 幸好這並不是太大問題，  因為 Shot Note App 的下方位置有好幾個功能鍵， 可以將圖像的質素調節調節， 光暗、對比都可以調。





而且右下角更有按鍵可以將圖像上傳到Evernote 及Dropbox 作記錄， 又或是用Twitter 及eMail 作分享， 亦可以多存一份到cameara roll 以備不時之需或別的用途。



結語:

Shot Note 產品定位清晰， 亦能達到目的將兩種記事方式的優點共融使用， 令用家得到最大的好處。基本就是這樣， 簡簡單單的直接了當。

題外話， 朋友曾經提問那些特製的記事簿可否自製?? 我相信是可以的， 但我的看 Shot Note 是在推廣一整個的手寫藝術數碼化的解決方案， 賣的是美感與滿足感， 我想朋友不要省那三數十元的零錢跑去麻煩自己， 最後只怕弄得又醜又無糟。
