【專欄】Blogger Event 怎樣搞系列 - 宣傳概念的改變

(續 : Blogger Event 怎樣搞系列 - 時間有限?)

“宣傳概念的改變 － 長尾效應”
談到商機，筆者相 信是沒有即時的回報吧，雖知道如上述的，弄一個體驗會最多都是三數十個博客，小則更是小貓十數隻，能賺的金錢都只是有限，相信大型的品牌公司不會著眼於這 些零零碎碎的回報上。即是說體驗會沒有什麼商機可言，所以筆者提意品牌公司明白到自身的口碑不是一朝一暮就可以建立的事情，所以倒不如經博客作推廣並與傳 統媒體的宣傳配合，建立Word of mouth口耳相傳的銷售網絡，拓展、尋求新的方式獲得新客戶。
// // Y.E.onDOMReady(show_notes_initially);

回說，其實體驗會不妨可安排得像 傳統的目標小組（Focus Group)
，找一些特定的博客來試產品的某些特點，而會後則可以再找參與了的博客作文章介紹。若以找30個博客來試用產品為例，就算當中只有三份之一人 會作出試用報告共十多篇文章，而每一個博客所寫的可能只有一日500人次閱讀到，但加起上來就會變成一日5000人次看過這個產品的介紹，看上來這些數字 並不吸引，但這些全是被讀者高度信任的介紹及評論。雖然隨著時間過去，這些文章於搜尋器的排名會往後推，但只要在網上搜尋則可以即時找得到，而博客所寫的 文章正是這樣地符合長尾效應的理論－持久性及由少眾集合力量，不像傳統媒體如報紙、雜誌，只一次性地向十萬名讀者介紹產品，而實際上很多時人們看過後便會 忘記或把雜誌扔掉，令到這些產品介紹不會長遠流存。

所以在概念上品牌公司及公關公司的人員請清楚地明白到博客的文章之價值，不是單重於其 瀏覽量，而是其影響力及可信任性，請不要再滿足於一般的瀏覽數字當中。博客的文章會在互聯網裡長存，所以其影響力的本質與傳統傳媒的大大不同，不要將兩者 時空倒置。
