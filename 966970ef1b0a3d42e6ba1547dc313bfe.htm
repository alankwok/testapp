香港的NIKE iD



上年中的時候到了日本大阪希望探訪日籍友人及看看好像Discovery Channel中的日式祭神慶典”天神祭”， 但最後朋友忙及我們懶， 友人沒有探， 卻四處閒逛發現了慾去已久的大阪Nike專門店， 因它有提供著 NIKE iD 服務 - 沒有太多國家有的自選材料製作的 NIKE 球鞋服務。



 

早兩星期發現了香港剛有NIKE iD 服務提供， 但當時尚未正式啟用， 所以只好預定時間再來。

等了一個星期， 終於來到約會的晚上， 一看見這個 NIKE iD 系統的頁面， 我們已經興奮起來， 馬上選擇中國， 因為香港的暫時未能用， 先要相就一下， 而當中的價錢單位會以人民幣作計算， $100的人民幣亦算為$100元的港幣收費， 所以變相是給予了香港區的朋友八八折。再馬上去簡單看看選購的程序。





紫色的是另一國家寄來香港NIKE iD 作樣辨用，所以香港可能不一定做到相同的配置。



上圖灰色的我十分喜歡，是Air force one 的款式，但可惜香港的iD 沒有此灰色提供。店內的陳列架更有二三十對同款的排列著， 何其壯觀， 但全都是用作人客試鞋尺寸之用。而那款鞋的鞋跟位置特別有一組數字約似 m10 / w8這樣， 用意標示那鞋男女裝分別是什麼尺寸大小。



香港的NIKE iD 提供給Air Force One Low Cut 的其中一種材料是漆皮，上圖是店內的樣辨，顏色真的很吸引，低iD 的價錢實在太貴，就好像買了一對Bathing Ape的一樣，當然BA 沒有AIR。

另一對好像比較只合農曆新年時穿著，放在店可能是給自由的朋友買來慶祝新年吧!



而我就是選了這對玩滑板款式的波鞋。



首先看清物料的真身，店員有幾套給我們慢慢比較慢慢看的。



再於系統中選出各部份的材料及顏色配置，每一款式都有不同的材料選擇，即是說的要的6.0 有紅色的皮， Air force One 的款式是不會有的，Kobe 5 有的，Air Force 亦不會有。所以拼湊出來的鞋，可能不是最想有的那一對呢！



上圖就是我的設計，最後我將NIKE 的標誌改成了紅，但鞋舌的灰色就改不了，因為那部份沒有紅的。



再下一步就可以加一個自選的英名字到鞋身，字亦可改顏色，但最多只有8個字母，而且有一些字眼是不能使用，例如BearBear 那對起初希望鍵入bearbear的，但都給拒絕了。如不喜歡加字， 就可找回 "NIKE" 標誌。



試穿一下尺寸，就可以選擇自己合用的下單！亦快將完成整個程序。



所有東西都決定好了，店員會幫我們印一張訂單，再去旁邊的桌面付款就是了。因為想收到時都各有驚喜，所以特別地將兩對鞋子分開了單據付款，那就可以分別寄到我與BearBear的公司地址。經查問寄的方式都是用FedEx ／ DHL 等運送，會於星期一至五工作時間內，如果找不到收件人就會容後再上門一次，再找不到就會到了倉庫中，等有關人員再跟進呢!



單據可基本看到鞋的各樣資料。（這對橙色的是bearbear自製的)
