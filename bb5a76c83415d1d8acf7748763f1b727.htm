【活動】Windows 8 VIP party

上星期獲邀出席了 Microsoft 為新版本 Windows 8 舉辦的Party， Party 原意是向合作伙伴介紹新系統的功能， 但有幸獲安排與一眾博客友好參加是次party， 實在是一個好好的機會， 輕輕鬆鬆四處玩玩，聽聽microsoft 同事的講解，總算是了解多了。就讓我介紹一下當晚的情況及談談對 Windows 8 Prof 及 Windows 8 RT 的看法。


Party 於跑馬地的 Adrenaline舉行， Adelina 都是Clubbing 的地方， 早前都已經去過亦有介紹過， 當晚一進場的位置就已經有一個大型的展位， 主要是介紹是 Windows 8 的 新 Microsoft Office 功能， 而展出的notebook 硬件用上了hp 用上win 8 的新型號。

【The New Office】 Microsoft即將推出全新Office，雖說是新的， 主要都是內裡的小功能改進了不少， 但不是有新軟件加入到office 套裝當中。例如其主打的電郵及日厝軟件 Outlook， 新的版本就已經支援了Touch Screem 觸控形式，  只需輕觸螢幕虛擬鍵盤便會即時彈出，用戶輕鬆快捷地回覆電郵。另一主打 Word 文書處理軟件更有全新的閱讀模式， 讓用戶在 Windows 8 的平版電腦 (tablet) 上用觸控手勢 (gesture) 閱讀各種文件。如果用Powerpoint， 亦同樣可以輕鬆在螢幕上縮放那 ppt，比舊有用滑鼠及鍵盤的使用模式快捷得多了。



Windows 8 裡面都有一個近年大熱的功能， 自家內置的 Store， 說明了就是介紹及提供下載軟件的功能， 相信大家都不難了解， 很多手機系統都已經有了。

香港的 MS 更與本地應用程式開發商合作，為Windows 8 設計出很多精簡易用的本地便民應用程式，例如: 香港的股票行情、香港的天氣報告、電影情報等資訊性的。而且更與一些商業機構合作，就好似於各大技術平台都積極擴展的中原地產、海洋公園、中華煤氣等等， 同時興多家公司合作， 看上來可算是與時並進， 跟過往Microsoft 大紅大熱的作風很類近，就好像感覺到Microsoft 的雄風要回來了一樣， 很有霸氣的王者。





另外一個角落， Microsoft安排了插畫師幫嘉賓以手上的Windows 8 Slate 繪畫人體素描，完成後更示範了用win 8 的新功能 Share Charm 透過 email (其他社交媒體都可以支援) 送回給嘉賓，看上來win 8 多了一些 社交媒體及分享功能的整合， 十分之方便。

   



而party 就當然要有酒精!! 是嗎!? win 8 的party當中便特意做了一個軟件給嘉賓選擇雞尾酒之用， 這原仕是一個商業應用的實例， 據知 Red Mr 卡拉ok 公司的 Surface 觸控臺於過往的兩年，都幫助了他們公司不少， 人客都愛玩他們的小遊戲及下單功能， 而且更帶來了不少傳媒的留意及報導，即是証明了好玩的商業應用應有其效用。

         



另外場內都有以家庭遊戲作主打的介紹， 雖然玩家於近年都好像轉了玩手機遊戲及專業的家庭遊戲機較多， 但當然在Win 8內，除了可幫助工作，亦是一向地可自行安裝喜愛的game玩。 不得原提的是， 原來 win 8 現在支援了 Xbox的遊戲， 玩家可以在win 8 的notebook / tablet 上玩到如angry bird space 等， 而且有些xbox 遊戲更會將小部份內容搬移到運行 win 8 的當中， 令到玩家出外的時候都可以不停地升 Level ， 回到家中就再用 xbox 同步玩下去， 以RPG game 的模式來說， 情況就好像拿著win 8 notebook 出外儲經驗值， 回到家中就利用早上儲回來的經驗玩xbox 上大型的遊戲一樣， 是一個新的遊戲模式呢!! (其實文字很難表達）， 相信這個是Windows 8 於遊戲方面 的最強之處。





Party 上 MS 也安排了一個結他手與鍵琴手利用touch screen 的Apps 合奏 jam 歌，效果出奇的好， 繞樑三日， 可見 Win 8 的音效控制功能都是有水準之作。  



當日同場其實有十多二個電腦廠商在展出新的windows 8 機種， 各種設計都看來有很多的突破， 很 Cool 的樣子， 特別是一些可拆鍵盤的tablet 混合體何其吸引，我對這些很有好感， 只是有些品牌的產品實在定價太貴， HK$7000 以上的產品多的是 :(








【Windows 8 Prof   <v.s.>  Windows 8 RT】

整晚與 Microsoft 的同事多談了，對Win 8 prof 及 RT 的版本都可叫了解多一點， 兩者當中microsoft 的宣傳上其實都沒有刻意地解釋分別在於那裡，如果看其他網站的介紹， 看了好幾篇其實仍然有摸不著頭腦的感覺。


《Windows Prof》

據我所了解 Prof 版本的就是我們過往的時候用的 Windows 系統， 是一個全面原整的大型系統， 跟對家的 Mac OS 是同一情況的應用， 當然地兩者細分之下是有很大的分別， 但概括地說， 可說是電腦用的系統好了。所以Prof 版是完全地能夠運行所有大小型軟件， 小型的意思就是RT 的版本。



《Windows RT》

而 RT 其實硬件上較為輕便的手提裝置所使用的系統， 如果以對家的Apple 來比較解說一下， 概念上可能是 iPad 等級的 iOS 差不多的定位。當中可運行的軟件， 就只支援 RT 的版本的， 一些大型的軟件於 RT 系統上是用不來的。



【軟件應用上去了解 Win 8 Prof 及 Win RT 的分別】

換個角度去看， RT 版本的 Windows 8 就只有一些如手機上的 "美圖秀秀" 等級的類型。


而Prof 版的 Win 8 是可運行 "美圖秀秀" 等級的類型， 與及 Photoshop 、 A.I. 及 AutoCad 等巨大功能的軟件， 亦可同時運行。這樣說大家應該會明白多一點。


最後都有一個要留意的地方， 不要只單靠硬件外觀上去認定系統的類型， 因為行 Windows 8 Prof 版的也有筆記薄型電腦， 亦有tablet 型的， 也有 tablet 型再加可拆式鍵盤的。


而運行RT 版本的， 基本上都會有上述的幾個配置， 所以選購時要先看清楚系統版本的分別。
