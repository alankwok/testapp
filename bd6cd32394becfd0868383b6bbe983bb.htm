【攝影】Lomo Matrix 大旅行


::

早前寫了Lomography  Summer Course 的第一堂課，今天說說課堂二 - Lomo大旅行，旅行中我們集體創作一個聽說很好玩的短片 - LOMO Matrix， 若大家有看過奇洛李維斯的 Matrix 都應該對一個鏡頭有印象， 奇洛李維斯站在街上彎腰避開飛來的子彈， 鏡頭便圍著他拍攝， 那個拍攝的效果早年給一班用Lomo 的朋友複製了過來， 亦成為了 Lomo 的其中一種玩樂點子， 亦再改了個名字叫 Lomo Matrix， 如果大家上youtube 搜尋一下， 應該能找到不少其他國家的Lomo Matrix 短片。↓ 而當日我們都拍了一段，大家可以看下面的，相信你會認得出我。

::


::



::


::

在拍攝之先，Lomo 的店長弟發了一些顏色鮮豔的手工紙給我們，不知怎的啤啤居然能夠摺出妻多不同的公仔出來，說賓話真的有點像七月十四燒的那些，但其實我們是用來作拍攝時的道具，多一點七彩的小手作於手上，拍出來的相片會更為富色彩，這就是Lomo 的特色之一。


::
其實Lomo matrix 這種玩樂形式，用數碼相機拍攝反而未必能有好效果，原因是按下數碼相機快門那刻，其實並不是即時拍攝，而只是作自動對焦及作其他準備，所以這並不能即時攝下中間正在做古怪動作的那位傻人，隨非改用手動對焦那就會好一點，但可惜的是有手動對焦功能的數碼相機並不普遍。此情況反而用Lomo 這些菲林相機更好，因為一按快門就即時攝入影像，沒有了時間的遲延所以更為合適。
