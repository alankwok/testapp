免費申請雲端儲存PCCW uHub

上兩星期臨起行到福岡旅行的早一天，PCCW 的公關朋友剛好預備了一個新的多媒體雲端服務給我試玩，就是最近大家都會見到廣告的uHub 服務， 什麼是多媒體雲端服務？什麼又是uHub? 簡單地， 大家可以說成是一個連接到internet就可以用到的外置硬碟空間，大家有什麼影片、相片、音樂、文件檔等等等等多媒體都可以放上去。

而 uHub  的自家特色就是可以幫忙將大家各種不同格式的檔案， 轉換成可隨時播放的串流格式， 將此優點加進了服務當中， 那就不再是網上儲存空間那麼簡單了。

讓我快跑一個申請步驟， 先拿下一個免費戶口， 再另開文章測試評論它的好與壞、快與慢?!

↓ 進入申請網址中已經看見簡單表達的支援類型， 影片、相片、音樂、文件檔， 我最著眼於影片的儲存， 其實就是貪心uHub 能夠幫忙轉換影片的格式， 跟據他們網站所提到共支援的格式包括3GP、AVI、FLV、M4V、MOV、MP4、MPG、MPEG、RM、RV、RMVB、WMV、M2TS、MTS、TS和MKV， 真係水蛇春那麼長的列表。



::
↓ 開始測試之前細看介紹， 下圖的步驟一的右下角便可 "立即登記"



::
↓ 到了步驟二就會發現於香港很少見到的情況， 就是有一選項 "我不是網上行寬頻及電訊盈科流動通訊用戶" ， 莫非有香港公司願意提供免費午餐給大家享用!?




::

↓ 步驟三就要大家填入個人資料正式申請uHub 的戶口， 再申報手機號碼就大致填妥， 但要留意的是申請uHub 為什麼需要填報手機號碼? 就是因為容後的時間系統會自動SMS 一個啟動碼給大家， 所以切勿亂填一通!! 緊記



 

::

↓ 步驟四就再多看一次確認一下資料有沒有錯誤， 對好了便按"確定"



::

↓  到了步驟五， 就是上圖提到要填入啟動碼， 這時你應該查看手機有沒有收到 SMS， 找到了便將內裡的啟動碼填到網頁當中



::

↓  弄好了網頁便會跳轉到最後的步驟六， 這時戶口基本上已經開啟了。本身若不是電訊盈科客戶則可有1GB免費容量;  現有的電盈服務用戶將獲免費5GB容量;  10月31日前申請電盈服務如網上行寬頻亦會獲免費升級至25GB， 詳情請自行查詢。

再說， 版面的下方同時亦提供連結下載Windows XP， Windows Vista， Windows 7 都合用的uHub桌面軟件， 唯獨是有點美中不足的是 Mac OS 的版本仍未見提供， Mac OS 的用家可能要多等一下了。

題外話， 申請完成 PCCW uHub 戶口，  他們會給你一個 xxxx@netvigator.com 的新ISP電郵地址， 大家大可以用來再申請高登的討論區， 變相方便了申請成為高登的巴打、絲打。有趣~



::

↓  另外亦有提供到手機系統Apple iPhone 及 Android的版本， 即是現時主要的兩大用戶群都已經照顧到， 相信大部用家都已經能夠滿足。



::



::

↓ 最後看看uHub 的介面， 戶口狀態及用量都一目了然， 清楚看到自己剩下多少空間。



::

總結:

申請簡單快捷， 免費1GB 用量可能太少， 但供一般用家免費申請， 則是創了香港Cloud 服務的先河。多手機系統支援亦是賣點之一， 亦破了久不更新的懷印象， 值得讚賞!!
