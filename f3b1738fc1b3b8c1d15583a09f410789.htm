【手機】one2free 4G LTE 測試報告

幾個星期前接受了公關公司的轉介與One2Free 公司接上了，因此參觀了CSL 於旺角新開的旗艦店， 現在旗艦店的位置原先是中南圖書用品的舊位置，但最近他們搬去了，而位置就轉了給CSL 租下來。此店有兩層，地下一個小地方放了幾幾個櫃檯，一樓全層則有齊CSL 公司各樣服務的介紹，而且更與 Suzuki Cafe 合作提供客人玩機、充電及等候的好地方，這算是香港近年才有的服務模式。

今次參觀One2Free 旺角旗艦的主要目的當然不是嘆咖啡，而是看看最新第四代流動數據服務4G LTE的示範，體驗一下4G 的速度。



 

先引述 Wikipedia 有關4G 技術及定義等介紹給大家了解：
4G 指的是流動電話系統第四代，也是3G之後的延伸，是一個成功的無線通訊系統。

按照技術標準的角度看，按照ITU的定義，靜態傳輸速率達到1Gbps，用戶在高速移動狀態下可以達到100Mbps，就可以作為4G的技術之一。

從營運商的角度看，除了與現有網絡的可兼容性外，4G要有更高的數據吞吐量、更低時延、更低的建設和運行維護成本、更高的鑒權能力和安全能力、支持多種QoS等級。

從融和的角度看，4G意味着更多的參與方，更多技術、行業、應用的融合，不再局限於電信行業，還可以應用於金融、醫療、教育、交通等行業；通訊終端能做更多的事情，例如除語音通訊之外的多媒體通訊、遠端控制等；或許局域網、互聯網、電信網、廣播網、衛星網等能夠融為一體組成一個通播網，無論使用什麼終端，都可以享受高品質的資訊服務，向寬帶無線化和無線寬帶化演進，使4G滲透到生活的方方面面。
 
可能成型的4G標準

WiMAX（全球互通微波存取）：由美國Intel所主導，是目前傳輸距離最遠的4G技術也是目前發展最快技術，在移動通訊環境下可以讓下行與上行最高速率各可達到75Mbps及75Mbps；新一代的IEEE 802.16m (WiMAX 2)可讓行動接收下行與上行最高速率可達到300Mbps，在靜止定點接收可高達1Gbps。[1]2008年11月12日HTC與俄羅斯流動通訊電業者Scartel（品牌名為Yota）共同發表全球第一支GSM／WiMAX整合式雙模手機HTC Max 4G。


LTE Advanced：是LTE的增強，完全向後兼容LTE，通常通過在LTE上通過軟件升級即可，升級過程類似於從WCDMA升級到HSPA。峰值速率：下行1Gbps，上行500Mbps。


回說當晚，其實我們於現場測試了幾個方式的下載，例如看Stream 的Video 等，因為大多數的情況下都尚算順利，所以都沒有拍下什麼特別的照片，下面就是其中一張我們試著下載一個 50m 的音樂檔，即時測試亦只需要20秒時間就完成。相比現時用3G連接方式下載真的快了好幾倍之多，感覺當然地良好。



 

亦因為我們借了one2Free 他們的 LTE 4G 的USB modem 流動數據機回家測試三個月之久，所以現場的時候我都只顧普跟友人閒聊，等回家才慢慢測試過夠吧。



 
開箱
。。。。。。時間過得很快，我已經回到家中試試安裝 One2Free的4G LTE 流動數據機，開箱文式介紹－一個小小的包裝盒就是印著LTE 制式的 4G ，速度達至100 Mbps ，正是付合wikipedia 當中提到的標準。



 

包裝盒內裡有4G LTE USB  流動數據機一個，香港人俗稱它為USB上網手指



 

盒的後面印著相關的資料，支援  4G LTE 2600、UMTS 900/2100、EDGE 900/1800/1900 各個制式

作業系統則支援 Windows XP SP2、 Windows Vista、Windows 7、Mac OS 10.6



 

盒內另有快速入門指南、USB 延長線及一年的保用証一張。



 
簡易安裝、支援Mac 系統
聽說之前一代的數據機沒有這代的方便，最少我現在將它插進我的Macbook Air 再經過簡單程序就已經安裝完成，不用另外下載驅動程式等步驟，頗方便的。



 

當安裝好了就會自動接上4G網絡，亦同時會彈出下面這個視窗顯示出 Data Connected 數據接駁著這提示。



 

如果希望再分享出來給其他機一同使用，就好像平常一樣於"互聯網分享 / Internet Sharing"功能中找找USB Ethernet 就可以了，沒有什麼特別的困難。



 
速度測試
正正式式作速度的測試，其實我覺得沒有什麼好多談的，因為數據都是死物，顯示出什麼就是什麼，沒有錯誤的空間，而且USB  modem 都是隨錯即用的，沒有好像十多年前的電腦硬件一樣，因為安裝錯了驅動程式所以有表現不穩的問題。

總結：

測試了好幾個地方的數據都沒有出現100Mkps 的下載速度，我實在有點奇怪，就好像是裝錯了驅動程式，可能今次的測試是基於 Mac OS，測出來的數值就好像不是用4G LTE 。 回想當晚在One2Free 店測的時候用Windows 系統， 測各樣方式都很順暢， 又真的比3G 的好上幾倍的表現是非常吸引的， 所以今次好像有點問題的測試， 我很希望日後有機會可再多試一次。

若根據其他朋友於windows 上的測試數據， 與及以服務費平貴來作論點， 其實4G LTE是能夠與3G 服務比較一下，因為如今one2free的4G LTE  服務月費又不是太貴，5GB的月費價錢都是HK$287， 比3G 的只是貴了少少，價錢上仍然是可接受的水平，大家其實應該要對都可留意一下，  但用家們應先自行問問朋友借4G LTE 的作實地測試， 如果接駁良好，4G 會是大有作為。當然地如果無限任用計劃的月費可下調至二百多元的水平會更為吸引， 。
